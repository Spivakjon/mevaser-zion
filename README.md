## סקירה כללית

מבשר ציון היא אפליקציית ניהול קהילת בית כנסת הבנויה כ-**Progressive Web App** (PWA). המערכת מאפשרת:

- **רישום וניהול מתפללים** — פרטים אישיים, משפחה, ילדים, יארצייטים
- **לוח שנה שבועי** — פרשות, חגים, בני מצווה, יארצייטים, ימי הולדת עבריים
- **זמני תפילה** — הדלקת נרות, צאת שבת, שחרית, מנחה, ערבית (חישוב מקומי לתל מונד)
- **תורנויות** — רוטציה הוגנת אוטומטית עם החלפות ידניות
- **עליות לתורה וקידוש** — שיבוץ שבועי
- **חברות ותשלומים** — מעקב, תזכורות WhatsApp, הנפקת קבלות
- **ייצוא/ייבוא** — JSON, CSV/Excel, vCard (אנשי קשר)
- **סנכרון ענן** — Google Sheets (אופציונלי)

### טכנולוגיות

| רכיב | טכנולוגיה |
|-------|-----------|
| Frontend | Vanilla HTML5, CSS3, ES6+ JavaScript |
| Backend | Google Apps Script (אופציונלי) |
| אחסון | localStorage + Google Sheets |
| זמנים | [KosherZmanim](https://github.com/KosherJava/zmanim) v0.9.0 |
| לוח עברי | [Hebcal API](https://www.hebcal.com/home/developer-apis) |
| פונטים | Google Fonts — Heebo + Frank Ruhl Libre |
| פריסה | GitHub Pages |

## טופס רישום

### שאלת דמי חבר (Prompt לפני הטופס)
לחיצה על "📝 הרשמה" פותחת תחילה כרטיס שאלה:
> "אנו שמחים לרשום את פרטי המתפלל לשנת **{שנה}**. האם שולמו דמי החבר לשנה זו?"

- **כן, שולם** → פותח את טופס ההרשמה
- **עדיין לא** → מציג כרטיס הסבר על הוגנות ושוויון בקהילה + כל קישורי התשלום הקיימים (שנתי / מופחת / חד־פעמי) + כפתור **אסדיר את התשלום בהמשך ←** שממשיך לטופס

השנה נלקחת מ-`mbz_membership.year` (ניתן להגדרה דרך הגדרות מנהל → חברות). ברירת מחדל: `תשפ״ו`.

### שדות חובה
- שם פרטי, שם משפחה, טלפון
- תעודת זהות (עם ולידציית Luhn mod 10 — רק ברישום חדש)

### שדות נוספים
- שם האב, שם האם
- **שם לעלייה לתורה** — מתמלא אוטומטית: "שם בן אב למשפחת משפחה"
- דוא"ל
- **כתובת מובנית**: רחוב, מספר בית, דירה, ישוב (ברירת מחדל: תל מונד)
- **בן/בת זוג**: שם, טלפון, דוא"ל

### ילדים (עד 7)
לכל ילד:
- שם, מין (זכר/נקבה)
- **תאריך לידה** — עברי או לועזי עם המרה דו-כיוונית (Hebcal API)
- **בר/בת מצווה** — חישוב אוטומטי:
  - 13 שנה לבנים, 12 לבנות
  - מציאת פרשת השבוע הקרובה לתאריך
  - שדות: פרשה, קריאה בתורה (כן/לא), הפטרה (כן/לא)

### יארצייטים (ללא הגבלה)
לכל יארצייט:
- שם הנפטר
- תאריך פטירה (עברי/לועזי)
- לפני התיבה (כן/לא), הפטרה (כן/לא)

### אפשרויות נוספות
- סוגי התנדבות (checkboxes — ניתנים להתאמה)
- הצטרפות לרשימת תורנויות

### הגנות
- **Rate limiting** — מקסימום 3 רישומים לשעה (sessionStorage)
- **בדיקת כפילויות** — לפי ת.ז.
- **ולידציית ת.ז.** — אלגוריתם Luhn ישראלי

## ממשק ניהול

**סיסמה**: `Nir2026`

### דשבורד סטטיסטיקות
7 כרטיסים: משפחות, ילדים, יארצייטים, בני מצווה קרובים, מתנדבים, שילמו דמי חבר, תורנים.

### טאבים

#### 1. 📅 לוח שבועי (Calendar)
- 12 שבועות עם פרשות, חגים, אירועים
- ניווט קדימה/אחורה/היום
- **קפיצה לתאריך** — עברי או לועזי
- כל שבוע מציג: בני מצווה, יארצייטים, ימי הולדת
- קישור לקריאת הפרשה (חב"ד)
- מתכווץ אוטומטית במובייל

#### 2. 📋 מתפללים (Members)
- רשימת חברים עם חיפוש טקסט חופשי
- **סינון**: הכל | מתנדבים | יש ילדים | יש יארצייט | תורנים | שילמו | לא שילמו | תגיות מותאמות
- **מיון**: שם | טלפון | ילדים | התנדבות | תשלום | תורנות
- **כרטיס חבר מפורט**: פרטים אישיים, בן/בת זוג, חברות, תגיות, ילדים, יארצייטים
- עריכה / מחיקה / שיוך תגיות / toggle תורנות ותשלום
- **ייבוא**: Excel (CSV), vCard (אנשי קשר), JSON
- **ייצוא**: Excel (CSV עם BOM לעברית), JSON (העתקה / טאב חדש — תואם iOS)

#### 3. 📜 עליות וקידוש (Aliyot)
- 8 עליות: כהן, לוי, שלישי...שביעי, מפטיר
- שדה autocomplete לבחירת חבר מהרשימה
- **קידוש**: מי מארח + הערות
- נשמר לשבוע הנוכחי ומוצג בדף הציבורי

#### 4. 📢 מודעות (Announcements)
- הוספה / עריכה / מחיקה
- כותרת + תוכן
- מופיע בלוח המודעות הציבורי

#### 5. 💳 חברות (Membership)
- **הגדרות**: שנת חיוב, קישור לתשלום
- **מעקב**: שולם/לא שולם + תאריך, אמצעי, סכום, הערות
- **סינון**: הכל | שילמו | לא שילמו
- **תזכורות WhatsApp**:
  - שליחה בודדת (חוב / חידוש)
  - שליחה מרוכזת לכל מי שלא שילם
  - **תבניות עריכה** — ניתן לערוך את נוסח ההודעה
  - משתנים: `{name}`, `{year}`, `{link}`
- **מידע חברות**: מוצג בדף הציבורי (סכומים, אמצעי תשלום, פרטי יצירת קשר)

#### 6. 📋 תורנויות (Duty Roster)
ראו [מערכת תורנויות](#מערכת-תורנויות).

#### 7. 🕯 יארצייטים (Yahrzeits)
- טבלה מרכזית של כל היארצייטים
- **סינון לפי חודש עברי**
- עמודות: חבר, שם הנפטר, תאריך עברי, תאריך לועזי, חזן, הפטרה

#### 8. 🙏 מתנדבים (Volunteers)
- רשימת כל המתנדבים עם סוגי ההתנדבות
- טלפונים ליצירת קשר

#### 9. 📊 טבלה מלאה (Full Table)
- תצוגת טבלה רחבה עם כל הנתונים
- **סינון**: הכל | מתנדבים | יש יארצייט | בר מצווה
- גלילה אופקית עם sticky headers
- עמודות דינמיות (ילדים, יארצייטים — לפי מספר מקסימלי)

### הגדרות (Settings)
- **לוגו**: העלאה / צילום / הסרה (JPG/PNG/WebP, עד 500KB)
- **טקסטים**: שם הארגון, כותרת משנה, טקסט קבלת פנים, כפתור הרשמה
- **סוגי התנדבות**: הוספה / מחיקה / שינוי שם (מותאם אישית)
- **תגיות**: גבאי, חזן, בעל קורא, ועד — הוספה / מחיקה / שינוי שם
- **זמני תפילה יומיים**: toggle להצגה/הסתרה בדף הציבורי

## מערכת תורנויות

מערכת תורנויות חכמה המבוססת על **רוטציה הוגנת**:

### איך זה עובד
- כל חבר שסימן "רוצה להיות תורן" נכנס לתור
- בכל שבת משובצים **2 תורנים**
- האלגוריתם ממיין לפי: מי שהיה הכי פחות פעמים → מי שהכי הרבה זמן לא היה

### תכונות
- **כרטיס hero** — תורנים לשבת הקרובה עם טלפונים
- **טבלת 8 שבועות** — ניווט קדימה/אחורה (עד 32 שבועות)
- **החלפה ידנית** — לחיצה על שם → בחירת מחליף
- **המלצה חכמה** — מי הכי צריך להיות? (לפי הוגנות + 12 שבועות)
- **+1 אורח** — שיבוץ אדם שלא ברשימה עם autocomplete
- **הסרה מ-slot** — ניקוי משבצת בודדת
- **איפוס לרוטציה** — ביטול swap ספציפי
- **סדר אוטומטי** — מיון מחדש לפי הוגנות
- **סיום תורנות** — סימון שבוע כבוצע + הסטת תור
- **חגים** — שיבוץ אוטומטי לערבי חג (פסח, סוכות, שבועות, ר"ה)
- **שורות מותאמות** — הוספה ידנית של תאריכי תורנות
- **מילוי חכם** — auto-fill לכל המשבצות הריקות
- **היסטוריה** — מעקב אחר כל השבועות שעברו
- **סנכרון Sheets** — queue, history, swaps

## סנכרון Google Sheets

### התקנה

1. פתח [script.google.com](https://script.google.com) → פרויקט חדש
2. הדבק את תוכן `apps-script.gs`
3. **Deploy** → **New deployment** → **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
4. העתק את ה-URL
5. בממשק הניהול → "הגדר חיבור" → הדבק את ה-URL

### API Endpoints

**GET:**
- `?action=getAll` — מחזיר את כל המתפללים כ-JSON
- `?action=getDuty` — מחזיר נתוני תורנויות
- `?action=ping` — בדיקת חיבור

**POST:**
- `{action: "save", member: {...}}` — שמירת חבר חדש
- `{action: "update", member: {...}}` — עדכון חבר
- `{action: "delete", id: "..."}` — מחיקת חבר
- `{action: "saveDuty", duty: {...}}` — שמירת תורנויות
- `{action: "sync", members: [...]}` — סנכרון מלא

## PWA ועבודה אופליין

### Service Worker (`sw.js`)
- **Cache**: `mbz-v1` — precache של index.html, logo.jpg, manifest.json, Google Fonts
- **API calls** (Hebcal): Network-first — מנסה רשת, נופל ל-cache
- **נכסים סטטיים**: Cache-first — משתמש ב-cache, נופל לרשת
- **Offline fallback**: מחזיר index.html מה-cache לניווט

### manifest.json
- שם: "קהילת מבשר ציון"
- מצב: standalone (ללא chrome של דפדפן)
- כיוון: RTL
- צבע רקע/ערכנות: `#1a2744`

### התקנה
- **Android**: כפתור "התקן" אוטומטי (beforeinstallprompt)
- **iOS**: הנחיית "שתף → הוסף למסך הבית"
- **Desktop**: תפריט דפדפן → "התקן אפליקציה"
