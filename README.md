- **כפתורים עליונים**: עריכת פרטים | ניהול | מצב בהיר/כהה | עב/EN
- **לוגו** — תמונה מ-`logo.jpg`
- **כרטיס שבועי** — פרשת השבוע + זמנים + אירועים:
  - 📖 פרשת השבוע (מ-Hebcal API)
  - 🕯️ הדלקת נרות + ✨ צאת שבת (קואורדינטות תל מונד)
  - ✡ בר/בת מצווה השבוע
  - 🕯 יארצייטים השבוע
  - 🎂 ימי הולדת עבריים
  - 📜 עליות לתורה (אם הוגדרו)
  - 🍷 קידוש (אם הוגדר)
- **מודעות** — מלוח המודעות של ממשק הניהול
- **כפתור הרשמה** — פותח את טופס הרישום
- **מידע על הקהילה** — אקורדיון ממוזער עם 12 סעיפי מידע
- **טופס רישום** (מוסתר כברירת מחדל)

### טופס רישום
שדות:
- שם פרטי*, שם משפחה*, שם האב* (חובה)
- שם האם, שם לעלייה לתורה (מתמלא אוטומטית), טלפון*
- דוא"ל
- כתובת: רחוב, מספר בית, דירה, ישוב (ברירת מחדל: תל מונד)
- ילדים: שם פרטי, מין (זכר/נקבה), תאריך לידה (עברי/לועזי)
  - רישום לפרשת בר/בת מצווה עם חישוב אוטומטי (13 לבנים, 12 לבנות)
- תחומי התנדבות: קניות לקידוש, סידור קידוש, דבר תורה
- יארצייטים: שם הנפטר, תאריך, לפני התיבה, הפטרה

### עריכת פרטים עצמית
מתפלל קיים יכול לערוך את פרטיו:
- זיהוי: טלפון + שם מלא + שם האב
- לאחר זיהוי — הטופס נטען עם הנתונים לעריכה

### ממשק ניהול
סיסמה: `Nir2026`

**טאבים:**
1. **📅 לוח שבועי** — 12 שבתות עם פרשות, אירועים, בני מצווה, יארצייטים
   - ניווט לתאריך (עברי/לועזי)
   - "קפוץ לתאריך" מתכווץ במובייל
2. **📋 מתפללים** — רשימה עם חיפוש + סינון מתקדם
   - סינון: הכל | מתנדבים | יש ילדים | יש יארצייט
   - "צפיה" → כרטיס מידע מסודר (לא טופס עריכה)
   - מהכרטיס: עריכה / מחיקה
3. **📜 עליות/קידוש** — שיבוץ עליות לתורה (8 עליות) + ניהול קידוש
4. **📢 מודעות** — לוח מודעות (הוספה/מחיקה, מוצג בדף הראשי)

**דשבורד סטטיסטיקות** בראש ממשק הניהול:
- סה"כ מתפללים, ילדים, יארצייטים, בני מצווה קרובים, מתנדבים

**ייצוא:**
- JSON (העתקה/טאב חדש — תואם iOS)
- CSV/Excel עם BOM לעברית
- שחזור מגיבוי JSON

## פיצ'רים טכניים

- **קובץ יחיד** — HTML + CSS + JS בקובץ אחד (~2100 שורות)
- **ללא תלויות** — לא צריך npm/build, רק דפדפן
- **רספונסיבי** — 3 breakpoints: 768px (טאבלט), 600px (מובייל), 380px (קטן)
- **תואם iOS** — font-size 16px, min-height 44px, safe-area-insets, viewport-fit=cover
- **מצב כהה/בהיר** — toggle עם שמירה ב-localStorage
- **דו-לשוני** — עברית + אנגלית (RTL↔LTR), data-i18n attributes
- **שם לתורה אוטומטי** — "שם פרטי בן שם האב למשפחת שם משפחה"
- **בר/בת מצווה** — 13 לבנים, 12 לבנות, חישוב פרשה אוטומטי
- **חודשים עבריים** — מיפוי וריאציות Hebcal (Iyyar, Sh'vat, Adar)
- **Google Fonts** — Frank Ruhl Libre + Heebo

## שינויים נפוצים

### שינוי סיסמת מנהל
שורה ~834: `const PW='Nir2026'`

### שינוי ישוב ברירת מחדל
חפש `value="תל מונד"` ושנה

### שינוי קואורדינטות לזמני שבת
חפש `latitude=32.258&longitude=34.917` ושנה

### הוספת שדה לטופס
1. הוסף input ב-HTML (בתוך `<div id="fv">`)
2. הוסף לפונקציה `submitForm()` — אסוף את הערך
3. הוסף ל-`initForm()` — טען את הערך בעריכה
4. הוסף ל-`showMemberDetail()` — הצג בכרטיס צפיה
5. הוסף ל-`resetForm()` — אפס

### הוספת תרגום
1. הוסף `data-i18n="myKey"` ל-element ב-HTML
2. הוסף את המפתח ל-`I18N.he` ו-`I18N.en`

### הוספת מידע קהילתי
חפש `iacc-` — כל סעיף הוא `<div class="info-acc">` עם head ו-body
