# מבשר ציון — מערכת רישום חברים

אפליקציית web לרישום מתפללים לבית כנסת קהילת מבשר ציון.  
קובץ HTML סטטי יחיד — ניתן לפתוח ישירות בדפדפן או לפרוס ב-Netlify.

---

## קבצים

| קובץ | תיאור |
|------|--------|
| `index.html` | האפליקציה המלאה — טופס רישום + ממשק מנהל |
| `apps-script.gs` | קוד Google Apps Script לחיבור Google Sheets |

---

## הפעלה מקומית

פשוט פתח את `index.html` בדפדפן (Chrome / Firefox).  
**לא עובד ב-Safari iOS מקומית** — יש לפרוס ב-Netlify.

---

## פריסה ב-Netlify

1. כנס ל-[app.netlify.com/drop](https://app.netlify.com/drop)
2. גרור את `index.html` לתיבה
3. לחץ **Rename** כשישאל על `index.html`
4. קבל URL — שלח למתפללים

---

## חיבור Google Sheets (אופציונלי)

1. פתח [sheets.google.com](https://sheets.google.com) → גיליון חדש → קרא לו **"מבשר ציון"**
2. תפריט **תוסף → Apps Script**
3. מחק הכל → הדבק את `apps-script.gs`
4. **Deploy → New deployment → Web app**
   - Execute as: Me
   - Who has access: Anyone
5. לחץ Deploy → אשר הרשאות → **העתק URL**
6. ב-`index.html`: ממשק מנהל → כרזה צהובה → "הגדר עכשיו" → הדבק URL

## פרטי מערכת

| פרמטר | ערך |
|--------|------|
| סיסמת מנהל | `Nir2026` |
| אחסון מקומי | `localStorage` (מפתח: `mbz_v5`) |
| אחסון Sheets URL | `localStorage` (מפתח: `mbz_sheets_url`) |
| לוח שנה | Hebcal API (`i=on` — לוח ישראל) |
| פרשות | `/hebcal` עם `s=on&maj=on&ss=on` |

## פיצ'רים

- טופס רישום מתפללים עם שדות: שם, משפחה, אב, אם, תורה, טלפון, אימייל, כתובת
- ילדים ללא הגבלת כמות — כל ילד עם תאריך לידה + רישום אופציונלי לפרשת בר/בת מצווה
- חישוב אוטומטי של פרשת בר מצווה (לידה + 13 שנה עברית → שבת + פרשה)
- יארצייטים עם בחירת תאריך עברי/לועזי + המרה אוטומטית
- ממשק מנהל עם:
  - לוח שבועי 12 שבתות עם פרשות, חגים ושבתות מיוחדות
  - ניווט לתאריך (עברי / לועזי)
  - רשימת מתפללים עם עריכה, מחיקה, חיפוש
  - סנכרון Google Sheets
  - ייצוא JSON
